# (Bataille)

## Description
Ce programe consiste a joeur a la bataille navale

## Génération
Vous avez besoin de:
 
- Windows 10
- Clion

Pour générer l'exécutable, vous devez:

1. Ouvrez de fichier Clion qui est dans le repository
1. Compilez le code
1. Sauvegardez le fichier dans un dossier

## Exécution


1. Allez dans le Dossier où vous avez sauvegardé le fichier 
1. Ouvrez l'executable
1. Vous Pouvez jouer
